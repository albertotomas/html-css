O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar páginas web. Ele estrutura o conteúdo, como textos, imagens, links, tabelas, vídeos, formulários, etc.

HTML não é uma linguagem de programação – ele não tem lógica, como "if", "for", "funções", etc. Seu papel é estruturar a informação e dizer ao navegador o que é o quê.
